让 iOS 与 Windows 实现剪贴板共享

2022-03-20

clipboard-online 是一款可以帮你在 💻Windows 和 📱iOS 之间分享剪切板的应用

项目地址

  • Github/clipboard-online

  • 不止于剪贴板，支持图片媒体互传
    iPhone 通过相册内捷径快速将图片媒体传输到 Windows

Windows 端

  1. 下载 clipboard-online
  2. 放在常用磁盘内，首次运行会在目录下生成 temp/config.json/log.txt
  3. 修改 config.json 配置文件
{
  "port": "8086",
  ## 修改authkey项的token
  "authkey": "token",
  "authkeyExpiredTimeout": 30,
  "logLevel": "warning",
  "tempDir": "./temp",
  "reserveHistory": false,
  ## 修改通知推送
  "notify": {
    "copy": true,
    "paste": true
  }
}

iOS 端

类型 链接 二维码
从 PC 复制到 iOS https://www.icloud.com/shortcuts/f463a1e431c94c60b8a5c65305eb819f
从 iOS 粘贴到 PC https://www.icloud.com/shortcuts/90e7a2af70df4707a17dece8c263afc5
  1. 添加捷径到你的 iPhone 里。
  2. 修改捷径、在 词典 中添加本地局域网 SSID & PC IP
  3. 修改文本框内为上方配置文件中 authkey 项的 token
