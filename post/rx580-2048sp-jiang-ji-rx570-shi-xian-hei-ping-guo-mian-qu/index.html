在 RX580 2048sp 刷入 570 VBIOS 实现黑苹果免驱

2020-05-22

RX580 2048SP 版本并不是完整的 RX580 ，没办法直接免驱黑苹果，但是可以刷 570 的 VBiOS 让系统识别为 570，进而做到免驱显卡。

  • AMD 蓝宝石RX580 2048SP 4g 刷显卡bios降RX570 黑果免驱
  • AMD 蓝宝石RX580 2048SP 刷显卡bios降RX570 黑果免驱—细节补充

显卡特性

类 型 参 数 备注
盈通 RX580 2048sp 大地之神 8G dvi接口*2 版本 新版显卡是dvi*1 / DP*3

刷入 Vbios

  1. 进入 Atiflash 文件夹右击打开终端或命令行执行
    cd C:\Users\Administrator\Desktop\Atiflash // 替换你下载的 Atiflash 文件夹路径
  2. 依次执行下方命令
amdvbflash.exe -unlockrom 0   // 解锁显卡BIOS
ROM Unlocked   // 返回意思：BIOS 解锁成功
amdvbflash -f -p 0 XXXX.rom    // 替换要刷入的 ROM 名称
Restart System To Complete VBIOS Update    // VBIOS 升级成功 重启完成
  • ROM 需要放在刷 Atiflash 文件夹目录下！
  • 重启后如遇设备管理器不显示显卡，更新驱动即可。
